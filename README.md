# Redis Highway Repository

This repository contains the source code for `@koala42/redis-highway`, a high-performance Redis stream-based queue for Node.js.

## Packages

### [@koala42/redis-highway](./packages/redis-highway)
The core library offering:
- **High Throughput**: Optimized Lua scripts and pipelining.
- **Consumer Groups**: Efficient load distribution and isolated retries.
- **Reliability**: Auto-claim for crash recovery and Dead Letter Queue (DLQ).
- **Observability**: Built-in Prometheus metrics.
